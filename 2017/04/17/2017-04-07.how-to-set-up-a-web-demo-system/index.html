一、关于 web demo 展示平台

demo 描述技术原理最形象直观的工具，一个好的 demo 能帮助我们经常地回忆技术细节，从而督促我们查漏补缺。同时 demo 也是一个很好的分享工具，在分享和讨论的同时能够更好地提高技术水平。

本文的目的就是要和大家一起搭建这样一个基于 github page 的 demo 展示平台。

本文假设你已经：

  • 了解 git 基本操作
  • 拥有 github 账号并且在 github 上创建过项目
  • 有基本的 web 技术基础
